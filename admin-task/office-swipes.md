Admin Task : Office Swipes
===========================


**Programming Fobs**

1.  On the wall unit next to the fridge enter the code _1985E_
2.  Wait until the green light comes on then press _*(UserNo)E_ where the UserNo is the nth programmed fob number (e.g. UserNo 009 is the 9th fob programmed, next fob programmed will be 010 and so on)  
    NOTE: refer to the fob access spreadsheet for the list of users and fob numbers: S:\\SavvyPlus\\Office Swipes.xlsx
3.  Swipe the fob on the card reader on the front door
4.  Return to the wall unit and press _E_ to finalise the programming
5.  Press _E_ on the wall unit again to exit program mode
6.  Update the user number and fob number in the above spreadsheet
