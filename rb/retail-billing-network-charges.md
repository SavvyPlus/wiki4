Retail Billing : Network Charges
================================



Network charges:

*   Cover cost of transporting power from the generation source to the customer
*   Regulated by AER
*   Collected by retailers but passed on to the distribution businesses
*   Set on an annual basis
*   Intended to be cost-reflective allowing a return for the network business etc.
